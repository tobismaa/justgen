# Transaction Generator App

This is a standalone Next.js project created outside the original repo.

## What it does

- Takes a customer name and generation settings
- Generates statement-style transaction rows across multiple months
- Optionally inserts salary every month
- Allows custom special transactions on exact dates
- Shows a preview table and supports CSV export

## Important note

Node.js and npm are not installed in this environment, so dependencies were not installed and the app was not run here.

## When Node is available

1. Run `npm install`
2. Run `npm run dev`
